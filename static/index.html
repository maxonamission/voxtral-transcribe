Instellingen

Maak een key aan op console.mistral.ai

Kies tussen Mistral API (standaard) of een lokale vLLM server

1
WSL2 installeren

Open PowerShell als Administrator en voer uit:

wsl --install

Start je PC opnieuw op na installatie. Bij de eerste keer WSL openen maak je een Linux-gebruikersnaam en wachtwoord aan.

2
vLLM installeren

Open WSL (typ wsl in een terminal) en voer uit:

curl -LsSf https://astral.sh/uv/install.sh | sh && source ~/.bashrc && uv venv ~/vllm-env --python 3.12 && source ~/vllm-env/bin/activate && uv pip install "vllm[audio]"

Dit installeert uv (package manager) en vLLM in een Python-omgeving. Duurt circa 5 minuten.

3
GPU-check

Controleer of je GPU zichtbaar is in WSL:

nvidia-smi

Als dit geen output geeft, installeer dan de NVIDIA CUDA drivers voor WSL.

Bepaalt hoeveel audio-context in het GPU-geheugen past. Lager = minder VRAM nodig.

Het model (~9 GB) wordt automatisch gedownload bij de eerste start. Of start handmatig in een terminal:

vllm serve mistralai/Voxtral-Mini-4B-Realtime-2602 --enforce-eager --max-model-len 8000

Snelle voorlopige tekst wordt automatisch vervangen door nauwkeurigere versie. Gebruikt 2x API-credits.
